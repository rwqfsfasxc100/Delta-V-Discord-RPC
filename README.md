# Delta-V-Discord-RPC
Client mod used to connect to the Delta V Discord RPC program

> [!WARNING]
> Requires the respective application to operate.
> See [it's repository page for more information](https://github.com/rwqfsfasxc100/Delta-V-Discord-RPC-Application)
>
> Due to limitations with the library used for Discord RPC, and a lack of hardware for testing on my own end, no OS-X support will be provided of current.
